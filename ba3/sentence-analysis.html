🔍 ANALYSIS METHOD
1READ the IPA
Read literally. No letters! Each symbol = that sound. Scan L→R, note anything ≠ RP.
▼
2CHECK RHYTHM
Schwa [ə] present?
YES → stress-timed (Inner Circle / AAVE)
NO → syllable-timed (NigE / IndE / SingE)
▼
3CHECK RHOTICITY
Post-vocalic /r/ pronounced?
YES → IrE, ScotE, SAE, IndE
NO → RP, GAusE, AAVE, NigE, SingE
▼
4CHECK CONSONANTS
TH → [t̪ d̪] = IrE
TH → [f v] = ScotE / Broad AusE
TH → [t d] = NigE / SingE / AAVE
TH → [tʰ] = IndE
[ʈ ɖ] retroflex = IndE
[ʔ] final = SingE / urban BrE
▼
5CHECK VOWELS
Use Wells' Sets (right →)
Mergers = key diagnostic
KIT=FLEECE? → NigE/SingE
Wide diphthongs? → GAusE
PRICE → [aː]? → AAVE
▼
6CHECK SYNTAX
If morphosyntax visible:
be habitual → AAVE
do be → IrE
particles (lah) → SingE
reduplication → NigE
▼
PREDICTABLE or NOT?
Matches theory → expected
Contradicts → hypothesis WHY
TEMPLATE per feature:
① Observe [IPA] in word X
② Name the phenomenon
③ Cite: Puskás Ch.X:Y
④ Predictable? → because...
⑤ If not → hypothesis

KILLER FEATURES — Instant ID
IrE: dental [t̪d̪] + clear /l/ + rhotic + do be
ScotE: dark /l/ everywhere + rhotic + might could
SAE: BATH=[æ] + rhotic + flap [ɾ] + no mergers
GAusE: wide FACE [æɪ] + MOUTH [æʊ] + h-drop + non-rhotic
AAVE: PRICE→[aː] + copula Ø + be habitual + done/béen/finna
NigE: no schwa + KIT=FLEECE + downdrift + reduplication
IndE: retroflex [ʈɖ] + no schwa + rhotic + isn't it
SingE: particles lah/mah + glottal [ʔ] + no schwa + A-not-A
