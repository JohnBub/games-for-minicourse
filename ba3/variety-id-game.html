How this game works

Every item is drawn from your Ihsane 2026 chapters — the same words, the same IPA, the same varieties you'll meet on the exam. Five modes train different exam skills:

  • Identify Variety — word + IPA shown, you pick the variety. Tests clip-to-label association.
  • Identify Feature — word + IPA + variety shown, you name the phonological rule. Tests terminology.
  • Minimal Pair — two transcriptions of the same word shown, you pick which variety each goes with.
  • Spot the Intruder — four features shown for a named variety, you pick the odd one out.
  • Signature Assembly — pick every feature that belongs to a named variety (multi-select).

Items you miss come back sooner (Box 1), items you get right move up (Box 2, then Box 3). Boxes 1 items appear with highest frequency in the rotation. This is a classic Leitner system.
